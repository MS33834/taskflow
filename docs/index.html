TaskFlow

本地优先、跨平台的任务管理工作流

项目简介

TaskFlow 是一个基于 React Native + Expo 开发的跨平台任务管理应用。 同一套代码可运行在 Web、Android 和 iOS 上，所有数据默认存储在本地 AsyncStorage， 无需账号、无需服务器即可离线使用。

仓库同时包含一个实验性的 Python FastAPI 后端，目前尚未与前端打通， 主要用于后续扩展服务端同步、插件等能力。

技术栈

React Native 0.73 Expo 50 TypeScript 5 Zustand 4 React Navigation v6 Python FastAPI

快速开始

# 前端
npm install
npm run web      # 浏览器预览
npm run android  # Android
npm run ios      # iOS（需 macOS + Xcode）

# 后端
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
